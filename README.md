## Project Overview
This project focuses on identifying and detecting safe landing zones for drones using advanced computer vision techniques.

## Dataset
The model was trained and evaluated using a custom dataset of drone-view images and videos, specifically focused on landing pad environments.

## Technical Details
- **Programming Language:** Python
- **Tools/Libraries:** OpenCV, TensorFlow, and NumPy.
- **Software Environment:** Visual Studio Code.

## Key Results
The system achieved high detection accuracy (over 90%) in identifying landing markers across various altitudes and lighting conditions.
