# Autonomous Drone Landing Detection

## Project Overview
This project focuses on identifying and detecting safe landing zones for drones using advanced computer vision techniques.

## Dataset
- **Data Source:** Custom dataset of drone-view images and videos.
- **Size:** 500+ annotated images.
- **Conditions:** Tested across various altitudes and lighting conditions.

## Technical Details
- **Programming Language:** Python
- **Tools/Libraries:** OpenCV, TensorFlow, NumPy.
- **Software Environment:** Visual Studio Code.

## Key Results
- **Accuracy:** The system achieved over 90% success rate in identifying landing markers.
- **Performance:** Optimized for real-time processing.
